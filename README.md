# candidate-insight

A mobile app that tracks political candidates information and presents it via infographics. The aim is to provide citizens with clear understanding of the leaders they are supporting.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/candidate-insight.git
cd candidate-insight
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
